## 1. 系统登录与退出

### 1.1 访问地址

打开浏览器，访问系统地址：

```
http://192.168.101.32:8266
```

### 1.2 登录

1. 在登录页面输入**用户名**和**密码**
2. 点击 **登录** 按钮
3. 验证通过后自动跳转至仪表盘主页

> **首次登录**：如管理员设置了强制修改密码，登录后自动跳转至修改密码页面。

### 1.3 退出

点击右上角用户头像 → **退出登录** 即可退出系统。

## 2. 仪表盘

仪表盘展示系统关键数据的可视化概览。

**功能：**

- 查看各类统计卡片（员工总数、部门数、待处理消息等）
- 图表展示（支持柱状图、折线图等）
- 可自定义仪表盘组件（需仪表盘配置权限）

---

## 3. 部门管理

管理企业组织架构，支持多级部门树。

### 3.1 访问路径

左侧菜单 → **部门管理**

### 3.2 操作说明

| 操作 | 说明 |
|------|------|
| **查看** | 以树形结构展示所有部门 |
| **新增** | 点击上级部门 → 点击"新增" → 填写部门名称 → 保存 |
| **编辑** | 点击部门节点 → 修改信息 → 保存 |
| **删除** | 点击部门节点 → 点击"删除" → 确认删除（如有子部门或员工，无法删除） |

---

## 4. 员工管理

管理企业员工基本信息。

### 4.1 访问路径

左侧菜单 → **员工管理**

### 4.2 操作说明

| 操作 | 说明 |
|------|------|
| **查询** | 按部门、姓名、状态等条件筛选 |
| **新增** | 点击"新增" → 填写信息（姓名、性别、部门、手机号等）→ 保存 |
| **编辑** | 点击编辑图标 → 修改信息 → 保存 |
| **删除** | 点击删除图标 → 确认删除 |

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | 是 | 员工真实姓名 |
| 性别 | 否 | 男/女 |
| 部门 | 是 | 所属部门 |
| 手机号 | 否 | 联系方式 |
| 邮箱 | 否 | 电子邮箱 |
| 岗位 | 否 | 职位名称 |
| 工号 | 否 | 员工编号 |
| 入职日期 | 否 | 入职时间 |

## 5. 用户管理

管理系统登录账户。用户通过角色获得权限。

### 5.1 访问路径

左侧菜单 → **用户管理**

### 5.2 操作说明

| 操作 | 说明 |
|------|------|
| **查询** | 按用户名、姓名等条件筛选 |
| **新增** | 点击"新增" → 填写用户名、密码、关联员工 → 分配角色 → 保存 |
| **编辑** | 点击编辑图标 → 修改信息/角色 → 保存 |
| **删除** | 点击删除图标 → 确认删除 |
| **重置密码** | 点击重置密码 → 设置新密码 |

### 5.3 密码规则

- 使用 BCrypt 算法加密存储
- 管理员可设置用户下次登录时强制修改密码

## 6. 角色管理

角色是权限的集合，用户通过角色获得权限。

### 6.1 访问路径

左侧菜单 → **角色管理**

### 6.2 操作说明

| 操作 | 说明 |
|------|------|
| **查询** | 按角色名称搜索 |
| **新增** | 点击"新增" → 填写角色名称和编码 → 分配权限 → 保存 |
| **编辑** | 点击编辑图标 → 修改信息和权限 → 保存 |
| **删除** | 点击删除图标 → 确认删除（如有用户关联该角色，无法删除） |

### 6.3 内置角色

| 角色编码 | 说明 |
|----------|------|
| super_admin | 超级管理员，拥有所有权限 |

## 13. 考勤管理

### 13.1 访问路径

左侧菜单 → **考勤管理** → 子菜单

### 13.2 考勤报表

查询和查看员工考勤统计。

**查询条件：**

- 员工（可选，不选则查全部）
- 开始日期 / 结束日期

**报表字段：**

- 员工姓名 / 部门
- 应出勤天数
- 正常出勤天数
- 迟到次数
- 早退次数
- 缺卡天数
- 节假日天数
- 费用合计

### 13.3 考勤员工管理

管理考勤系统中的员工信息。

**操作：**

- 查看考勤员工列表
- 编辑考勤员工信息
- **员工映射**：将考勤员工关联到系统员工（用于报表关联）
- 导出考勤员工列表

**批量映射：**

系统会自动根据员工姓名尝试匹配，未匹配的员工可手动关联。

### 13.4 打卡记录

查看和管理打卡记录。

**查询：**

- 按员工、日期范围筛选
- 分页展示

**操作：**

- 查看打卡记录详情
- 新增/编辑/删除打卡记录（手动补录）

### 13.5 数据同步

**同步员工（HWATT）：**

从 HWATT 考勤机数据库同步员工列表到系统。

**同步打卡记录（HWATT）：**

从 HWATT 同步指定时间范围的打卡记录。

**同步员工（钉钉）：**

从钉钉同步员工信息。

**同步打卡记录（钉钉）：**

从钉钉同步考勤打卡数据。

**导入进度：**

同步过程中，页面顶部显示实时进度条（基于 SignalR WebSocket）。

### 13.6 班次方案管理

管理考勤班次方案。

**操作：**

- 新增班次方案（名称、上班时间、下班时间、迟到宽限分钟数）
- 编辑/删除班次方案

### 13.7 排班时间管理

为员工配置排班时间。

**操作：**

- 按日期查看排班
- 批量创建排班计划
- 员工-方案关联管理

### 13.8 日类型管理

标记特定日期的工作类型。

**类型：**

- 工作日 (workday)
- 周末 (weekend)
- 节假日 (holiday)

### 13.9 节假日管理

管理法定节假日。

**操作：**

- 新增节假日（名称、日期范围）
- 删除节假日

### 13.10 费用计算器

配置考勤费用计算规则。

**操作：**

- 新增计算规则（名称、条件、公式）
- 编辑/删除规则

### 13.11 考勤数据权限

配置考勤模块的数据权限规则。

## 15. 常见问题

### 15.1 登录提示"用户名或密码错误"

- 检查用户名和密码是否正确
- 确认 Caps Lock 未开启
- 联系管理员重置密码

### 15.2 登录后提示"服务器内部错误"

- 检查后端 API 是否正常运行
- 查看后端日志定位具体错误

### 15.3 页面显示"无权限"

- 当前用户没有该功能的访问权限
- 联系管理员分配对应角色的权限

### 15.4 考勤数据同步失败

- 检查 HWATT 数据库连接是否正常
- 检查钉钉 API 配置是否正确
- 查看后端日志定位错误详情

### 15.5 导入进度卡住

- 检查 SignalR 连接是否正常
- 刷新页面后重试导入
