## 1. 系统登录与退出

### 1.1 访问地址

打开浏览器，访问系统地址：

```
http://192.168.101.32:8266
```

### 1.2 登录

1. 在登录页面输入**用户名**和**密码**
2. 点击 **登录** 按钮
3. 验证通过后自动跳转至仪表盘主页

> **首次登录**：如管理员设置了强制修改密码，登录后自动跳转至修改密码页面。

### 1.3 退出

点击右上角用户头像 → **退出登录** 即可退出系统。

## 2. 仪表盘

仪表盘展示系统关键数据的可视化概览。

**功能：**
- 查看各类统计卡片（员工总数、部门数、待处理消息等）
- 图表展示（支持柱状图、折线图等）
- 可自定义仪表盘组件（需仪表盘配置权限）

## 3. 部门管理

管理企业组织架构，支持多级部门树。

### 3.1 访问路径

左侧菜单 → **部门管理**

### 3.2 操作说明

| 操作 | 说明 |
|------|------|
| **查看** | 以树形结构展示所有部门 |
| **新增** | 点击上级部门 → 点击"新增" → 填写部门名称 → 保存 |
| **编辑** | 点击部门节点 → 修改信息 → 保存 |
| **删除** | 点击部门节点 → 点击"删除" → 确认删除（如有子部门或员工，无法删除） |

## 4. 员工管理

管理企业员工基本信息。

### 4.1 访问路径

左侧菜单 → **员工管理**

### 4.2 操作说明

| 操作 | 说明 |
|------|------|
| **查询** | 按部门、姓名、状态等条件筛选 |
| **新增** | 点击"新增" → 填写信息（姓名、性别、部门、手机号等）→ 保存 |
| **编辑** | 点击编辑图标 → 修改信息 → 保存 |
| **删除** | 点击删除图标 → 确认删除 |

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓名 | 是 | 员工姓名 |
| 工号 | 是 | 员工编号（唯一） |
| 性别 | 否 | 男/女 |
| 所属部门 | 是 | 从部门树中选择 |
| 职位 | 否 | 岗位名称 |
| 手机号 | 否 | 联系电话 |
| 邮箱 | 否 | 电子邮箱 |
| 入职日期 | 否 | 格式：yyyy-MM-dd |
| 状态 | 否 | 在职/离职 |

## 14. 报表管理

### 14.1 功能概览

| 功能 | 访问路径 | 说明 |
|------|---------|------|
| 报表列表 | 报表管理 → 报表列表 | 查看所有报表，支持分类筛选和排序 |
| 报表设计器 | 报表管理 → 点击"设计" | 在线设计报表（SQL/字段/过滤/排序/图表） |
| 报表查看 | 报表管理 → 点击"查看" | 执行报表、查看数据、导出 |
| 数据源管理 | 报表管理 → 数据源管理 | 管理报表数据源连接 |
| 销售员地区 | 报表管理 → 销售员地区映射 | 地图可视化展示 |

### 14.2 报表列表

- 左侧展示报表分类树
- 右侧展示分类下的报表
- 支持按更新时间排序
- 报表状态：已发布/草稿/已禁用
- 支持右键管理分类

### 14.3 报表设计器

#### 基本信息

| 字段 | 说明 |
|------|------|
| 报表名称 | 唯一标识 |
| 报表标题 | 显示标题 |
| 所属分类 | 从分类树中选择 |
| 数据源 | 主库(PostgreSQL) / ERP(SQL Server) |
| 状态 | 草稿/已发布/已禁用 |

#### SQL 编辑器

- 输入 SQL 查询语句
- 支持参数占位符（`@paramName`）
- 点击"测试"预览数据和字段结构
- 支持多数据源切换

#### 字段配置

- 自动从 SQL 结果读取字段列表
- 配置字段类型（string/number/date/boolean/money）
- 设置字段标题、宽度、对齐方式
- 配置是否可排序、可筛选、可分组
- 配置汇总类型（sum/avg/count/max/min）

#### 过滤条件配置

- 为参数添加输入控件
- 支持控件类型：文本输入、下拉选择、日期选择、日期范围
- 配置默认值

#### 排序配置

- 设置默认排序字段和方向
- 支持多字段排序

#### 图表配置

- 选择图表类型（柱状图/折线图/饼图）
- 配置 X 轴、Y 轴字段
- 设置标题和尺寸

#### 字段权限

- 按角色控制字段可见性
- 不同角色看到不同的列

### 14.4 报表查看

1. 在报表列表中点击"查看"
2. 输入查询参数（如有）
3. 点击"查询"执行报表
4. 查看结果表格：
   - 字段按设计器配置显示（类型、宽度、对齐）
   - date 类型显示为 `yyyy-mm-dd hh:mm:ss`
   - money 类型显示为千位分隔格式 `1,234.56`
   - 支持点击列头排序
   - 支持分页浏览
5. 支持导出 Excel（当前页 / 全部数据）

### 14.5 透视表

在报表查看页面切换到"透视表"标签：
- 将字段拖拽到行/列/值区域
- 支持计数、求和、平均值等聚合方式
- 支持保存透视表配置供下次使用
- 支持分享透视表给其他用户/角色

### 14.6 数据源管理

- 配置额外的数据库连接
- 支持 PostgreSQL 和 SQL Server
- 连接字符串加密存储

## 16. 常见问题

### 16.1 登录失败

**现象：** 提示"用户名或密码错误"  
**解决方法：**
1. 检查用户名是否输入正确
2. 检查密码是否输入正确（区分大小写）
3. 联系管理员重置密码

### 16.2 页面加载缓慢

**原因：** 报表查询数据量过大  
**解决方法：**
1. 缩小查询时间范围
2. 使用更精确的过滤条件
3. 联系管理员检查数据库性能

### 16.3 导出失败

**原因：** 数据量过大或网络超时  
**解决方法：**
1. 减少导出数据量
2. 使用分页导出
3. 检查网络连接

### 16.4 权限不足

**现象：** 页面按钮灰色不可点击或提示"无权限"  
**解决方法：**
1. 联系管理员分配相应权限
2. 确认当前用户角色配置正确
