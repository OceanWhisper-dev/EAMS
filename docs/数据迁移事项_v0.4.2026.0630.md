## 2. 迁移概述

本系统从 `v0.4.2026.0624` 升级到 `v0.4.2026.0630` **无需数据库迁移**。本次升级仅涉及前端依赖变更和代码层面的路由调整，数据库表结构保持不变。

---

## 3. 本次变更范围（无需迁移）

### 3.1 前端依赖变更

| 操作 | 包 | 说明 |
|------|-----|------|
| 移除 | `pivottable` | jQuery 透视表库 |
| 移除 | `jquery` | jQuery 核心库（仅用于 pivottable） |
| 移除 | `jquery-ui-dist` | jQuery UI（仅用于 pivottable 样式） |
| 新增 | `@visactor/vue-vtable` | VTable Vue 组件 |
| 新增 | `@visactor/vtable` | VTable 核心库 |

### 3.2 代码变更

| 变更 | 说明 |
|------|------|
| `ReportViewer.vue` | el-table/el-table-v2 → VTable ListTable |
| `PivotTable.vue` | pivottable (jQuery) → VTable PivotTable |
| `router/index.ts` | 系统管理路由统一为 `/system/*` 前缀 |
| 24 个 DTO 类 | 添加 `[Required]` / `[Range]` / `[MinLength]` 验证注解 |

### 3.3 数据库无变更

- 表结构不变
- 存储过程不变
- 索引不变
- 数据不变

## 4. 升级步骤

### 4.1 更新代码

```bash
# 拉取最新代码
git pull origin main

# 切换到新标签
git checkout v0.4.2026.0630
```

### 4.2 重新安装前端依赖

```bash
cd src/EAMS2026.Web

# 清理旧的依赖缓存
rm -rf node_modules
npm cache clean --force

# 重新安装（自动安装 @visactor/vue-vtable + @visactor/vtable）
npm install
```

### 4.3 重新构建

```bash
# 前端构建
npm run build

# 后端构建（如有配置文件更新）
cd ../EAMS2026.Api
dotnet publish -c Release -o ../../publish/api
```

### 4.4 重启服务

```bash
# 停止旧进程 → 更新文件 → 启动新进程
```
