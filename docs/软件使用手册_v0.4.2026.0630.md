## 2. 系统登录

### 2.1 登录方式

访问系统地址（如 `http://localhost:5107`），进入登录页面。

- 输入用户名和密码
- 勾选"记住密码"可保存凭据（localStorage）
- 点击"登录"按钮

> **注意：** 首次登录或管理员重置密码后，系统会强制要求修改密码。

### 2.2 登录后界面

登录成功后自动跳转仪表盘，页面结构：

| 区域 | 说明 |
|------|------|
| 顶部导航栏 | 左侧：Logo + 系统名称；右侧：消息铃铛（含未读计数）、用户头像下拉菜单（个人中心/修改密码/退出） |
| 左侧菜单栏 | 按模块分组的侧边导航菜单 |
| 主内容区 | 显示当前功能页面 |

## 3. 系统管理

### 3.1 仪表盘

路径：`/dashboard`

展示系统关键统计卡片和基于角色的仪表盘组件（由仪表盘配置功能设定）。

### 3.2 部门管理

路径：`/system/department` | 权限标识：`department`

- **部门树**以树形表格展示
- 操作按钮：新增（选择上级）、编辑、删除
- 新增/编辑弹窗包含：部门名称、编码、上级部门、排序

### 3.3 员工管理

路径：`/system/employee` | 权限标识：`employee`

- 列表展示员工信息
- 筛选条件：部门、关键字（姓名/工号）
- 操作：新增、编辑、删除（软删除）
- 高级功能：查看已删除列表、硬删除

### 3.4 用户管理

路径：`/system/user` | 权限标识：`user`

- 列表展示系统用户
- 操作：新增、编辑、删除（软删除）、重置密码、分配角色
- 新增用户时需关联员工，设置用户名和密码
- 分配角色：弹窗多选角色

### 3.5 角色管理

路径：`/system/role` | 权限标识：`role`

- 列表展示角色
- **角色权限分配**：树形权限勾选 → 保存
- **角色用户分配**：弹窗多选用户 → 保存
- 操作：新增、编辑、删除（软删除）

### 3.6 权限管理

路径：`/system/permission` | 权限标识：`permission`

- **权限树**以树形表格展示
- 权限类型：菜单、按钮、接口
- 操作：新增（选择上级）、编辑、删除
- 编辑时可设置路由路径和相关接口

### 3.7 字典管理

路径：`/system/dict` | 权限标识：`dict`

- 双面板：左侧字典类型 → 右侧字典项
- 左侧操作：新增、编辑、删除字典类型
- 右侧操作：新增、编辑、删除字典项（标签、值、排序、颜色、是否默认）

### 3.8 仪表盘配置

路径：`/system/dashboard-config` | 仅管理员

- 选择角色 → 配置该角色的仪表盘组件
- 组件类型：统计卡片、列表、快捷链接、信息卡片、图表
- 支持设置 SQL 数据源、刷新间隔、布局

### 3.9 消息管理

- 顶部导航栏铃铛图标查看未读消息
- 收件箱（`/message/inbox`）：查看消息、标记已读、回复、删除
- 已发送（`/message/sent`）：查看已发送记录
- 写消息（`/message/compose`）：选择收件人、填写标题/内容、设置优先级

### 3.10 操作日志

路径：`/system/operation-log` | 权限标识：`operation-log`

- 管理员查询所有用户的操作日志
- 普通用户仅查看自己的操作日志
- 筛选：用户、模块、日期范围

### 3.11 数据权限配置

路径：`/system/data-permission` | 权限标识：`data-permission`

- 选择角色，配置模块的数据权限范围（全部/所在部门）

## 4. 考勤管理

### 4.1 考勤报表

路径：`/attendance/report`

- 选择员工（支持搜索过滤）和日期范围查询
- 展示打卡记录和计算结果
- 支持导入打卡数据（有进度条）和打印报表

### 4.2 基础配置

| 功能 | 路径 | 说明 |
|------|------|------|
| 考勤类型 | `/attendance/day-type` | 定义日类型（工作日、休息等） |
| 排班类别 | `/attendance/scheme-class` | 管理排班方案 |
| 计划标准时间 | `/attendance/plan-time` | 设置排班时间 |
| 节假日 | `/attendance/holiday` | 按年份管理节假日 |
| 费用计算规则 | `/attendance/fee-calculator` | 配置费用计算 |

### 4.3 考勤员工管理

- 管理考勤系统员工与系统员工的映射关系
- 支持手动更新映射和导出

### 4.4 员工关联班次

- 将员工绑定到排班类别
- 支持新增、编辑、删除关联

### 4.5 事件声明

- 员工可声明加班、请假、出差、调休
- 管理员审批管理

## 5. ERP 辅助模块

### 5.1 报表管理

**报表列表** `/erp/reports`

- 查看所有可用报表
- 点击"预览"进入报表查看器

**报表查看** `/erp/reports/view/:id`

报表查看器功能：
- **数据表模式**：使用 VTable ListTable 渲染数据，支持列宽拖动、列排序、虚拟滚动
- **透视表模式**：使用 VTable PivotTable 渲染，支持拖拽配置行/列/指标
- **工具栏**：格式保存/加载、列设置、分享
- **汇总行**：自动计算的汇总数据
- **导出**：Excel / PDF

**报表设计** `/erp/reports/designer/:id`

- 配置报表的 SQL 查询、字段映射、过滤条件、排序、图表

### 5.2 单据修改

详见《ERP单据修改模块设计文档》。

| 功能 | 路径 | 权限 |
|------|------|------|
| 订单客户修改 | `/erp/vouch-order` | `erp-vouchmodify:order` |
| 发货单客户修改 | `/erp/vouch-dispatch` | `erp-vouchmodify:dispatch` |
| 发货日期修改（批量） | `/erp/vouch-date` | `erp-vouchmodify:date` |
| 修改日志 | `/erp/vouch-log` | `erp-vouchmodify:log` |

### 5.3 ERP 设置

| 功能 | 路径 | 权限 |
|------|------|------|
| 数据源配置 | `/erp/settings/datasources` | `erp-settings:datasource` |
| 业务员对照 | `/erp/settings/salespersons` | `erp-settings:salesperson` |

## 6. 常见问题

### 6.1 登录失败

- 检查用户名和密码是否正确
- 确认服务是否正常运行（后端:5106，前端:5107）
- 查看浏览器控制台是否有跨域或请求错误

### 6.2 报表数据不显示

- 确认数据源连接正常
- 检查 SQL 查询是否有效
- 确认当前用户有该报表的查看权限

### 6.3 路由跳转异常

- 系统管理模块的路由统一使用 `/system/` 前缀（如 `/system/user`，而非 `/user`）
- 如果使用旧的书签或手动输入 `/user` 路径，请更新为 `/system/user`
